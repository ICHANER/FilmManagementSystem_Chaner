## 影城管理系统
&#160;&#160;&#160;&#160; 本项目是一个影城管理系统，旨在帮助影院管理者有效管理电影放映、座位安排和票务销售等方面的任务。系统全部采用UI界面进行交互，提供了以下主要功能：
- 管理员、经理、用户、前台 四种用户类型；
- 用户管理：包括添加、删除、修改用户信息，以及创建删除用户；
- 电影管理：包括电影下映、排片管理等；
- 票务管理：票务销售与订单管理。

## 技术栈
- 开发工具：IntelliJ IDEA
- Java 版本： JDK 18.0.2
- 操作系统：macOS 14.0 (M1)
- 数据库：MySQL 8.2.0 - arm64
- 数据库连接：MySQL-Connector-J-8.2
- UI 框架：Java Swing
- 版本控制工具：Git
- 数据库管理工具：MySQL Workbench
